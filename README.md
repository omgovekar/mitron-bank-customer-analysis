# Mitron Bank Customer Analysis (Power BI)

## Project Overview
This project analyzes customer behavior and spending patterns for a fictional bank (Mitron Bank) using Power BI to support data-driven decision making.

## Objectives
- Understand customer demographics
- Analyze income utilization and spending behavior
- Identify high-value and potential customers
- Provide insights for better product and marketing strategies

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Pages
1. Customer Overview
2. Average Income Utilization
3. Spend Behavior Analysis
4. Potential Customer Identification

## Key Insights
- Identified high-income but low-utilization customer segments
- Observed spending trends across customer groups
- Highlighted potential customers for targeted offers

## Business Recommendations
- Target under-utilizing high-income customers with premium products
- Personalize offers based on spending behavior
- Improve engagement for low-activity segments

- Dataset is simulated for learning purposes
